LSM Sign Language Videos
========================
Place MP4 video files in this folder.

File names must match the "video_file" field in phrases.json.
Example: ph_001_hola.mp4, ph_016_gracias.mp4, ph_028_ayuda.mp4

Recommended specs:
  - Orientation : Portrait (vertical)
  - Resolution  : 480x854 or 720x1280
  - Codec       : H.264 (MP4 container)
  - Duration    : 1–4 seconds per phrase
  - Max size    : 5 MB per clip

The app works without any video files — it shows a placeholder icon.
Add videos at any time and rebuild via GitHub Actions.
